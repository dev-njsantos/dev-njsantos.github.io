This is an interesting project because we were assigned to make a User Interface for a "Smart Home" application and it is intended for users that have disabilities, to be specific, people with ALS. They are required to manage around their homes with this ease and this application should help them do that with the use of their wheelchair's sip and puff. This is our final project for our robotics class. Since there is no specified task that the robot has to do, the possibilities were endless on what we can build in this final project. We decided to make a Candy Sorter Robot which basically sorts out the colors of candies and put them in their respective containers while roaming around the table avoiding its own fall and stopping when it sees a person infront of it. We used a sensors such as RGB sensor for sorting the candy, Ultrasonic sensors and IR sensors to sense if the robot reached the edge of the table and if there is a person infront of the robot. We used a servo motor to move the dispensing arm to the proper container and a stepper motor to move the candies that are being scanned inside.
